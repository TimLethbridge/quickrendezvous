Rendez-vous rapide

Bienvenue dans le système de réservation de rendez-vous QuickRendezvous

Ceci a été conçu pour permettre la gestion d'énormes files d'attente où les gens ont besoin de rendez-vous rapides le plus tôt possible et qui durent relativement peu de temps. Les rendez-vous sont définis selon le principe du premier arrivé, premier servi.

Utilisations possibles:

  • Réservation de rendez-vous de test Covid-19 sur un site de test particulier (il est particulièrement adapté aux sites pop-up, il n'y a donc pas besoin de files d'attente physiques).
  • Réservation de consultations avec un enseignant ou un professeur (heures de bureau)
  • Réservation de vaccinations
  • Prise de rendez-vous dans les installations de loisirs, culturels out commerciels, où le nombre de personnes présentes à la fois doit être contrôlé.

Si vous disposez d'un lien de file d'attente, vous pouvez demander un rendez-vous en fournissant votre nom, votre adresse courriel et votre numéro de téléphone. Veuillez ne pas cliquer sur un lien pour utiliser une file d'attente à moins que vous ne fassiez confiance à la source du lien , car vous ne voulez pas donner vos informations à une source non fiable. En d'autres termes, suivez uniquement les liens vers les files d'attente de sites Web connus.

Le lien suivant vous mènera à une file d'attente d'entraînement où vous pourrez réserver des rendez-vous simulés et découvrir le fonctionnement du système.

Essayez la file d'attente d'entraînement.

Tout le monde peut cliquer ci-dessous pour créer une file d'attente et envoyer ou publier le lien. Le gestionnaire de files d'attente recevra un lien secret pour configurer les rendez-vous disponibles et voir les personnes qui ont réservé.

Créer une nouvelle file d'attente

Fonctions spéciales:

  • Les rendez-vous peuvent être définis pour n'importe quel nombre de jours dans le futur, mais les anciens disparaissent.
  • Les rendez-vous peuvent être générés automatiquement pour la même période de temps chaque jour ou pour des périodes différentes selon les jours.
  • De nouveaux rendez-vous peuvent être ajoutés par le gestionnaire de files d'attente (par exemple, si une capacité supplémentaire devient disponible).
  • Les gens obtiennent toujours le premier rendez-vous disponible par défaut, mais peuvent demander d'essayer de le changer pour un rendez-vous antérieur ou ultérieur. Par exemple, un rendez-vous antérieur peut devenir disponible en raison d'une annulation.
  • Ceci est conçu pour un temps de réponse très rapide.
  • S'il n'y a pas de rendez-vous, le système autorisera les demandeurs à réessayer ultérieurement.
  • Les rendez-vous peuvent être générés progressivement au fil du temps, évitant une précipitation (comme ce serait le cas si tous les billets de concert étaient libérés en même temps)
  • Le responsable des rendez-vous peut planifier plusieurs rendez-vous en même temps s'il le souhaite (par exemple pour une installation de loisirs).
  • Le nombre de rendez-vous disponibles à différentes périodes dans le futur est visible aux utilisateurs.
  • Le gestionnaire de files d'attente peut intégrer son propre logo et texte dans sa page.

Extensions prévues:

  • Vérification de l'adresse courriel ou du numéro de téléphone avant qu'un rendez-vous puisse être réservé (non gratuit).
  • Possibilité pour le gestionnaire de files d'attente de spécifier le type d'informations à collecter lorsqu'une personne demande un rendez-vous.
  • Rendre le site plus joli (sans compromettre sa vitesse).

Ce logiciel est open source sur GitHub et peut donc être installé sur n'importe quel serveur privé.
